How to Use DocIntel

DocIntel is an AI-powered document analysis API. Upload any PDF, Word document, or image and get an instant summary, extracted entities, and sentiment analysis. Below are the credentials and steps to get started.

Using This Website
1

Wake the API

Go to the Analyze tab. The endpoint and key are pre-filled. Click ⚡ Test Connection — the status dot turns green when the server is ready. If it takes 30s, that's normal on first cold start.

2

Upload a document

Drag and drop or click to browse. Supported: PDF, DOCX, DOC, PNG, JPG, JPEG, WEBP. Max recommended size: 5 MB. Image-based (scanned) PDFs are supported via OCR.

3

Click Analyze Document

The file is encoded and sent to the API. Text PDFs process in ~5s. Scanned/image PDFs use OCR and take ~60–90s. DOCX and images process in ~5–10s.

4

View results

The right panel shows: an AI-generated summary, named entities (people, dates, organizations, amounts), overall sentiment, and the raw JSON response. Click "Show JSON" to see the full API response.

Using the API Directly (cURL)
# Encode your file first BASE64=$(base64 -i your_file.pdf) curl -X POST https://sample-doc-analyzer.onrender.com/api/document-analyze \ -H "Content-Type: application/json" \ -H "x-api-key: sk_track2_987654321" \ -d "{\"fileName\": \"your_file.pdf\", \"fileType\": \"pdf\", \"fileBase64\": \"$BASE64\"}"
Request Body Format
{ "fileName": "invoice.pdf", // any name "fileType": "pdf", // pdf | docx | image "fileBase64": "JVBERi0xLjQ..." // base64-encoded file bytes }
Response Format
{ "status": "success", "fileName": "invoice.pdf", "summary": "This document is a tax invoice issued by OfficeCorp...", "entities": { "names": ["Ravi Kumar"], "dates": ["10 March 2026"], "organizations": ["ABC Pvt Ltd"], "amounts": ["₹10,000"] }, "sentiment": "Neutral" }
File Format Details

📄 PDF

  • Text-layer PDFs: fast (~5s)
  • Scanned/image PDFs: OCR fallback (~60–90s)
  • Multi-page supported
  • Invoices, reports, contracts

📝 DOCX / DOC

  • Paragraphs + table text extracted
  • Fast processing (~5s)
  • Resumes, agreements, letters

🖼️ Image (PNG/JPG/WEBP)

  • OCR via Tesseract
  • English, Hindi, Tamil supported
  • Receipts, screenshots, scans
  • Higher contrast = better results

🔐 Authentication

  • Header: x-api-key
  • Wrong key → 401 Unauthorized
  • Missing key → 422 error
